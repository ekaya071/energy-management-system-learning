## What I learned

### Git

It allows developers to:
- track changes,
- work together,
- return to previous versions,
- work on different branches.

---

### GitHub

---

### Virtual Environment (.venv)

A virtual environment creates an isolated Python environment for a single project.

Why?

Different projects may require different versions of the same package.

Instead of installing everything globally, each project has its own environment.

---

### Requirements File

A requirements file contains all Python packages needed for the project.

Example:

- numpy
- pandas
- matplotlib
- dash
- casadi

Installing them is as simple as:

pip install -r Requirements_PC_simulation.txt
